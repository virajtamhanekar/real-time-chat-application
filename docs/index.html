🔐 Real-Time Secure Chat App

End-to-End Encrypted Chat System using Python

🚀 Features

💬 Multi-user chat: Real-time messages with multiple users
🔐 Encryption: DES, RSA, and ElGamal secure messaging
🧾 TTP: Trusted Third Party-based authentication
🎛 GUI: Desktop interface with Python Tkinter

🖼️ Screenshots

🛠️ Tech Stack

  • Backend: Python
  • Frontend: Tkinter
  • Encryption: DES, RSA, ElGamal
  • Network: Python Sockets

📦 Installation

git clone https://github.com/AhmedIssa11/Real-Time-Secure-Chat-Application
cd Real-Time-Secure-Chat-Application

▶️ How to Use

python server.py      # Start server
python client.py      # Launch client (in separate terminals)
